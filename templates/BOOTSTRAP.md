# OpenClaw Superpack — Bootstrap Checklist

## Pre-Flight
- [ ] OpenClaw gateway installed and running
- [ ] Python 3.8+ available
- [ ] Node.js 18+ available
- [ ] Git configured

## Core Setup
- [ ] Mesh system initialized
- [ ] Health check passing for all enabled providers
- [ ] Cost routing configured
- [ ] Daily budget set

## Intelligence Pipeline
- [ ] Data sources configured in config.yaml
- [ ] First collection run completed
- [ ] Filter thresholds calibrated
- [ ] Digest delivery configured

## Agents
- [ ] Agent roster reviewed
- [ ] Selected agents activated
- [ ] Agent personalities customized (optional)

## Integrations (Optional)
- [ ] Telegram bot connected
- [ ] Twitter/X API configured
- [ ] WHOOP connected
- [ ] MCP servers configured

## Verification
- [ ] Run mesh health check
- [ ] Run intelligence collector test
- [ ] Verify agent communication
- [ ] Check cost tracking
