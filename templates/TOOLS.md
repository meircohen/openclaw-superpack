# Tools Reference

## Mesh Tools
- **router** — Classifies tasks and routes to optimal LLM provider
- **dispatch** — Executes tasks with fallback chains
- **health** — Monitors system health across all providers
- **stats** — Tracks usage, costs, and performance metrics
- **cost** — Calculates and optimizes LLM costs

## Intelligence Tools
- **collector** — Pulls from configured data sources (HN, GitHub, RSS, etc.)
- **filter** — Two-pass LLM classification pipeline
- **digest** — Generates markdown digests with priorities
- **act** — Autonomous action layer
- **trends** — Tracks patterns and emerging topics

## Delegation Tools
- **pipelines** — Multi-step automation workflows
- **templates** — Structured task schemas
- **verify** — Output verification checklists

## MCP Servers
- **ai-mesh** — Mesh system MCP interface
- **context-mode** — Semantic code search
- **context7** — Documentation lookup
- **claude-peers** — Multi-instance communication
