# MCP Servers Audit

Here is a list of the TOP 20 MCP servers we should ADD to our setup, based on the audit of the `best-of-mcp-servers` repository.

### Finance / Investment Tools

1.  **longbridge/openapi**
    *   **Description**: LongPort OpenAPI provides real-time stock market data, provides AI access analysis and trading capabilities through MCP.
    *   **Install Command**: `mcp install longbridge/openapi`
    *   **Usefulness**: This is a must-have for any finance-related work. It provides real-time stock market data and trading capabilities, which would be incredibly valuable for investment analysis and automation.
    *   **Priority**: must-have

2.  **polygon-io/mcp_polygon**
    *   **Description**: An MCP server that provides access to Polygon.io financial market data APIs for stocks, indices, forex, options, and crypto.
    *   **Install Command**: `mcp install polygon-io/mcp_polygon`
    *   **Usefulness**: This server provides a comprehensive set of financial market data, which is essential for any serious investment analysis. The variety of data sources makes it very versatile.
    *   **Priority**: must-have

3.  **base/base-mcp**
    *   **Description**: Base Network integration for onchain tools, allowing interaction with Base Network and Coinbase API for wallet...
    *   **Install Command**: `mcp install base/base-mcp`
    *   **Usefulness**: With the growing importance of cryptocurrencies, this server is a great addition for interacting with the Base Network and Coinbase, which are major players in the crypto space.
    *   **Priority**: nice-to-have
