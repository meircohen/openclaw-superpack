# Session Wrap-Up Protocol (from session-wrap-up skill)

At the end of every long session (>30 min or >10 tool calls):
1. Git commit any uncommitted changes
2. Extract key learnings → append to daily memory notes
3. Update pending_items.md with new tasks discovered
4. Update active_guidance.md if priorities changed
5. Generate a 3-line session summary for the daily digest
